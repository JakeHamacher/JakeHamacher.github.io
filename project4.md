## Project description

This project is a web-based POS system to be used in running specialty stores. It is built using Java/Vaadin(Java Framework) and MySQL.

## How to compile and run the program

The project is a standard Maven project. To run it from the command line, type mvnw (Windows), or ./mvnw (Mac & Linux), then open http://localhost:8080 in your browser.

You can also import the project to your IDE of choice as you would with any Maven project. Read more on how to import Vaadin projects to different IDEs (Eclipse, IntelliJ IDEA, NetBeans, and VS Code).

## UI Design

The program is pretty intuitive. There are different pages that allow the user to add or remove stock from the inventory, or to perform a purchase which gives a receipt and removes the appropriate stock from inventory.
